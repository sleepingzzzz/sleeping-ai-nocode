# Frontend Design

This skill provides comprehensive frontend design guidance for modern web applications.

## Core Capabilities

### UI/UX Patterns
- Design system implementation
- Component library patterns
- Accessibility (WCAG) compliance
- User interaction patterns
- Animation and transitions

### Component Architecture
- Atomic design methodology
- Component composition patterns
- Props and state management
- Slot and render prop patterns
- Higher-order components

### Responsive Design
- Mobile-first approach
- Breakpoint strategies
- Fluid typography
- Flexible layouts
- Container queries

### Styling Best Practices
- CSS methodologies (BEM, CSS Modules, CSS-in-JS)
- Theme implementation
- Dark mode support
- CSS custom properties
- Performance optimization

## Usage Guidelines

1. Always consider accessibility first
2. Follow mobile-first responsive design
3. Maintain consistent design tokens
4. Use semantic HTML elements
5. Implement proper focus management

## Framework Support

- Vue 3 (Composition API)
- React (Hooks)
- Angular
- Svelte

## Example Scenarios

- Creating a new component library
- Implementing responsive layouts
- Setting up design tokens
- Adding dark mode support
- Optimizing CSS performance
